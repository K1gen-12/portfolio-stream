# Personal Web Site (Blog + Lab)

このリポジトリは **Codex（AI）を使ってコードを追記・修正しながら育てていく個人Webサイト** です。  
完成形を一気に作るのではなく、**小さな変更を積み重ねて育てる**ことを前提にしています。

ブログ機能を持ちつつ、  
- 思考ログ  
- 個人開発の途中経過  
- 試行錯誤のメモ  

をそのまま残すための「個人ラボ」的なサイトです。

---

## コンセプト

> 作りながら考えたことを、そのまま置いていく場所

- 未完成でも公開する
- 途中の迷い・変更理由も残す
- 後から見返して「自分のOS」になることを重視

## 記事の追加方法

1. `content/blog` 配下に Markdown ファイルを追加
2. frontmatter を記述
3. 保存するだけで記事が追加される

### Markdown例

```md
---
title: "このサイトを作った理由"
date: "2025-01-01"
tags: ["thought", "dev"]
---
